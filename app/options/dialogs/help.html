Getting Started

Gestures is a Chrome extension for defining and performing mouse gestures. Analogous to Chrome's built-in keyboard shortcuts, gestures are specific sequences of mouse events that are assigned to actions; when you perform a known gesture, Gestures executes its corresponding action.

Example: To create a new tab using Chrome's keyboard shortcut you would press CTRL+T. To create a new tab using Gestures' default config you would press and hold the right mouse button, move the mouse down a short distance and release the right mouse button. The action in this example is New Tab and the gesture is MOUSE DOWN.

Gestures

Gestures supports two types of gestures, Mouse and Rocker, as well as Key Modifiers.

Mouse gestures are performed by pressing and holding a single mouse button (Right button in default config), moving the mouse in a series of cardinal directions to create (invisible) lines, and releasing the mouse button. The resulting sequence of lines is interpreted as the gesture.

Example: The default config assigns the simple gesture MOUSE DOWN to New Tab, and the more complex MOUSE DOWN+RIGHT+LEFT to Close All Windows.

Rocker gestures are performed by pressing and holding either the Left or Right mouse button and clicking the button that is not pressed (Right or Left). There are exactly two base Rocker gestures, RIGHT+LEFT and LEFT+RIGHT. Rocker gestures using mouse buttons other than Left and Right are not supported due to inconsistent behavior.

Example: The default config assigns the gesture ROCKER LEFT+RIGHT to the action Add To Reading List.

Key Modifiers are activated by holding one or more modifier keys, CTRL, ALT1, SHIFT and/or META2, when a gesture is triggered, i.e. on the button release for a Mouse gesture and on the secondary button click for a Rocker gesture. Key modifiers allow basic Mouse and Rocker gestures to be extended to create new gestures, thereby increasing the number of usable gestures without having to perform additional mouse movements.

Example: The default config extends MOUSE DOWN by adding the CTRL modifier, assigning the gesture MOUSE CTRL DOWN to the action Open Downloads.

1: ALT may be referred to as OPTION on some keyboards, e.g. Apple.

2: META is keyboard specific, e.g. Windows key or Apple Command key. META will rarely/never work on Windows systems due to the way the Windows key is handled by the operating system.

Targets

Gestures categorizes all actions as one of four types of targets: Generic, Link, Image and Selection. Link, Image and Selection actions will have Link/Image/Selection somewhere in the name of the action, e.g. "Download Image"; everything else is a Generic action. When a potential gesture is triggered each target type is processed to look for a matching target as well as a matching gesture. Targets are processed in priority order of Selection, Link, Image and finally Generic.

A Selection target is matched whenever there is a selected element (text) on the page. Unlike Link and Image, Selection does not require the gesture to start on the selected element.

A Link target is matched when the gesture starts1 on a link2.

An Image target is matched when the gesture starts1 on an image2.

A Generic target is matched by all gestures, including gestures that also match Selection, Link or Image.

Example: The default config assigns the gesture MOUSE DOWN to three different targets: Link, Image and Generic. If the gesture started on a link then the action New Background Tab From Link is executed; if the gesture started on an image then the action New Background Tab From Image is executed; and finally, if the gesture did not start on a link or target then the action New Tab is executed. Note that because MOUSE DOWN is not assigned to any Selection action the end result of the gesture is the same regardless of whether or not any element(s) on the page is selected.

Example: The default config assigns the gesture MOUSE CTRL DOWN to two different targets: Image and Generic. If the gesture started on a image then the action Download Image is executed, otherwise the action Open Downloads is executed.

1: Rocker gestures 'start' when the second mouse button is clicked.

2: Many images on the web are actually links represented by an image instead of text. These are treated as links by Gestures.

Options

Gestures automatically saves your options when any option is changed. Your options are also automatically synchronized across machines if you are signed into Chrome.

A snapshot of the options is called a config. At any given time only one config is active; the active config is referred to as the current config. The options page always reflects the current config. At the top of the page you will see 'Current config is Default config.', 'Current config is saved as ...' or 'Current config has not been saved'. This does indicate whether or not your options haven been saved/synchronized, but rather indicates whether or not the current config has been saved in the storage database with an assigned name so that the config can be retrieved later.

The Save, Load and Delete buttons allow you to save the current config into your storage database, load a config from your storage database, or delete a config from your storage database. As with all options, your saved configs are synchronized across machines when you are signed into Chrome. Above the Save/Load/Delete button you will see a brief blurb stating how many configs have been saved, and approximately how many additional configs can be saved in synchronized storage. Configs vary in size based on the number of gestures defined, which is why the number of additional configs that can be saved is an estimate.

The Export and Import buttons allow you to export the current config as JSON by copying it to your clipboard, and import a config by pasting its JSON representation into the import dialog. JSON is a text-based data format, which means that you can save an exported config in a text file, send it via email, and if you so desire, manually edit the config.

Options are organized by their types: switches, new tab URLs, and actions. The search bar in the upper right allows you to quickly search for an option by name. Clicking on the label (name) of any option will bring up a dialog that provides more information as to what the option controls or how an action is executed.

Switches are boolean options that control how Gestures behaves. The New Tab Url and New Incognito Tab Url options allow you to specify URLs that are used when opening a new tab via a gesture that does not have a predefined URL. A tab will have a predefined URL when it was opened to show a target link or image, or to navigate to a shortcut such as the Extensions page. Use chrome://newtab to load Chrome's New Tab page. Note that Chrome does not allow extensions to run on its Incognito New Tab page, even if the extension is given permission in the Extensions page. This means that using chrome://newtab as the URL for new Incognito tabs will prevent Gestures from working on new Incognito tabs, which is why the default config uses http://www.google.com as the New Incognito Tab Url.

Each entry/line in the actions options represents a gesture that is assigned to an action. The action label can be found on the left, followed by the gesture and buttons to edit, remove and add gestures for the action. Blank/missing action labels mean that multiple gestures are assigned to the above action. A blank gesture/edit/remove entry means there are no gestures assigned to the action. See below for a quick tutorial on how to read an action entry.

Stop
 This entry defines gestures for the Stop action.

ROCKERkeyboard_arrow_leftkeyboard_arrow_right     A Rocker gesture activated by holding the Left button and clicking the Right button.

keyboard_arrow_up keyboard_arrow_down keyboard_arrow_left keyboard_arrow_right      A Mouse gesture activated by moving the mouse UP, DOWN, LEFT and RIGHT while holding the mouse button.

CTRL ALT keyboard_arrow_down     A Mouse gesture activated by moving the mouse DOWN while holding the mouse button, CTRL and ALT.

edit
 Edit an existing gesture for an action.

 Remove an existing gesture from an action.

Tips

Hitting the forward-slash button anywhere on the options page will activate the search bar.

Click on the name of an option to learn more about that option.

Saving configs allows you to experiment with different gesture mappings without having to rebuild your preferred config from scratch.

All dialogs autofocus the confirmation button. You can simply hit ENTER to copy Exported settings or confirm an Import, Save, Load or Delete.

Rocker gestures 'start' when you click the second button. You can quickly open multiple links/images in background tabs by holding the Right mouse button and Left clicking on each link/tab you want to open.

Chrome does not allow extensions to run on certain sites; Gestures will not be active on these sites. Restricted sites include but are not limited to: chrome://* (except non-incognito chrome://newtab), chrome-extension://* and https://chrome.google.com/*.
